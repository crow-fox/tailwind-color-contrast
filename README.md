# Tailwind Color Contrast

TailwindCSS でデフォルトで設定されているカラーパレットのコントラスト比を計算するツールです。

## 開発環境

### 依存関係のインストール

```
bun install
```

### 開発サーバーの起動

```
bun run dev
```

### ビルド

```
bun run build
```

### ビルドしたファイルをローカルサーバーで確認

```
bun run start
```

### リント

```
bun run lint
```

### フォーマット

```
bun run format
```
